## 重置服务器
### 停止进程
```js
pkill -kill -u 
```
### 删除文件及其文件夹
```js
chmod -R 755 ~/* 
```
```js
chmod -R 755 ~/.* 
```
```js
rm -rf ~/* 
```
```js
rm -rf ~/.* 
```
